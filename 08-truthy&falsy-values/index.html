Truthy and Falsy Values in JavaScript

In JavaScript, values can be classified as either truthy or falsy.

Falsy Values:

  • false
  • 0 (zero)
  • "" (empty string)
  • null
  • undefined
  • NaN (Not-a-Number)

Truthy Values:

All values that are not falsy are considered truthy. Some examples include:

  • true
  • Any non-zero number (e.g., 5, -10)
  • Any non-empty string (e.g., "hello", "0")
  • Objects (e.g., {}, [])
