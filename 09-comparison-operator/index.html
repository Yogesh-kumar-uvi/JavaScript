Comparison Operators in JavaScript

Comparison operators are used to compare two values. The result of a comparison is either true or false.

Types of Comparison Operators:

  • == (Equality): Checks if two values are equal (type coercion is allowed)
  • === (Strict Equality): Checks if two values are equal and of the same type
  • != (Inequality): Checks if two values are not equal (type coercion is allowed)
  • !== (Strict Inequality): Checks if two values are not equal or not the same type
  • > (Greater than): Checks if the value on the left is greater than the value on the right
  • < (Less than): Checks if the value on the left is less than the value on the right
  • >= (Greater than or equal to): Checks if the value on the left is greater than or equal to the value on the right
  • <= (Less than or equal to): Checks if the value on the left is less than or equal to the value on the right
